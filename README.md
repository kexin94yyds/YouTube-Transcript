# 视频转 EPUB 项目

这是一个包含 YouTube 转录侧边栏和 EPUB 阅读器的综合项目。

## 项目结构

### YouTube-Transcript
YouTube 视频转录侧边栏 Chrome 扩展，支持：
- ✅ 自动获取视频字幕
- ✅ 点击时间戳跳转到视频对应位置
- ✅ 点击文本复制字幕片段
- ✅ 自适应窗口大小，填满整个高度
- ✅ 支持章节导航
- ✅ 搜索功能
- ✅ 导出 EPUB 电子书

### tobooks
EPUB 电子书阅读器，支持：
- ✅ 手势导航
- ✅ 高亮管理
- ✅ 用户认证
- ✅ 支付集成
- ✅ 书籍切割功能

## 安装和使用

### YouTube 转录侧边栏

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `YouTube-Transcript` 文件夹
6. 访问 YouTube 视频页面，点击扩展图标即可使用

### EPUB 阅读器

直接打开 `tobooks/index.html` 或部署到服务器使用。

## 功能特性

- 🎯 点击复制：点击字幕文本即可复制对应片段
- 📏 自适应大小：侧边栏自动填满窗口高度
- 🔍 搜索功能：在视频中搜索关键词
- 📚 EPUB 导出：将字幕导出为 EPUB 电子书
- 📌 固定模式：固定侧边栏，视频自动调整大小
