## 🎯 立即使用

### 方式1: 在线使用
1. 访问: https://tobooks.netlify.app
2. 选择任一私钥文件 (batch_keys/ 目录下)
3. 复制私钥内容到认证页面
4. 点击"验证身份" → 获得永久访问权限

### 方式2: 本地测试
```bash
cd /Users/apple/网页看书
PORT=3001 node server.js
# 访问: http://localhost:3001
```
